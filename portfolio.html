Employee Management System

October – December 2024

Python PostgreSQL

This is a console-based Employee Management System developed in Python with PostgreSQL as the backend database. It provides a complete solution for managing employee data and attendance within an organization. The system offers the following functionalities:

1. Add Employee – Register new employees by capturing essential details like name, department, designation, and contact information.

2. View Employee – Display all employee records stored in the database in a readable format.

3. Update Employee – Modify existing employee information based on user input.

4. Delete Employee – Remove an employee’s record permanently from the system.

5. Mark Attendance – Allow marking daily attendance (present/absent) for employees with timestamp support.

6. View Attendance Report – Generate daily/monthly attendance summaries for each employee.

7. Calculate Salary – Compute monthly salary based on basic pay, allowances, deductions, and attendance.

8. Generate Employee Report – Produce a detailed report for any employee, including personal info, attendance history, and salary data.

This system demonstrates the integration of Python (psycopg2 library) with PostgreSQL to perform CRUD operations, data validation, and basic reporting, making it a foundational project for learning full-stack software development principles.

Scramjet Combustor

January – April 2025

Ansys Fluent CFD Supersonic Flow

Enhancing Mixing in Supersonic Flow – A CFD Study of DLR Scramjet Combustor

This project focused on analyzing and optimizing fuel-air mixing and combustion efficiency in a DLR scramjet combustor using various strut-based fuel injector geometries (wedge, diamond, wavy, multi-strut, and a novel modified strut). Using ANSYS Fluent, cold and hot flow simulations were performed with compressible flow and turbulence-combustion models. Key parameters such as mixing efficiency, combustion efficiency, and flow behavior were evaluated. The modified strut design achieved the highest combustion efficiency (99.99%), proving its superiority for hypersonic propulsion.
